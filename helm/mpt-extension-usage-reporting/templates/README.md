Helm templates are generated automatically by ***template: helm/helm-init.yaml@ops-mpt-infra*** based on **./deployment-manifest.yaml** file.

Common templates can be found in [ops-mpt-infra](https://dev.azure.com/softwareone-pc/MPT/_git/ops-mpt-infra?path=/helm)

If you need special custom template it can still be placed in helm/{app}/templates directory.
